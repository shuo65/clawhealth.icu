Q: 如何安装OpenClaw？
  1. Windows用户：下载一键安装包，双击运行，按提示完成安装
  2. Mac用户：下载.dmg镜像，拖拽到Applications即可
  3. Linux用户：使用命令行安装：bash <(curl -fsSL https://install.openclaw.ai)
  4. 所有平台：安装后需要配置大模型API（如Step、GLM等）
Q: 配置API密钥时出错怎么办？
  • 检查API key是否正确（从官网获取，不是登录密码）
  • 确认网络连接，部分国内API需要代理
  • 验证API额度是否充足
  • 查看~/.openclaw/config.json配置格式是否正确
Q: 如何更新到最新版本？

推荐三种方式：

  1. 自动更新：打开OpenClaw，点击"检查更新"按钮
  2. 命令行：运行 openclaw update
  3. 重新安装：下载最新安装包覆盖安装（配置会保留）
Q: Agent运行缓慢怎么优化？
  • 减少同时运行的Agent数量
  • 使用更轻量级模型（如GLM-4-Flash、Step-1.5-Flash）
  • 增加系统内存分配（关闭其他占用大的程序）
  • 清理缓存：openclaw cache clean
  • 检查系统资源占用，避免CPU/内存瓶颈
Q: 如何备份和恢复配置？

备份：openclaw backup export（生成备份文件）

恢复：openclaw backup import <备份文件路径>

也可以手动复制 ~/.openclaw/ 目录到安全位置。
